Class: Game

Game

new Game( [config] [, state])

Game is the main class of creatine, it contains and initializes all systems for your games. It receives two parameters, a configuration object or an url for a JSON file, and an object containing state functions. Please consult {{#crossLinkModule 'creatine'}}this page {{/crossLinkModule}} for an overview of these parameters and the usage examples.
